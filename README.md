# Rock, Paper & Scissor's Game ✊✋✌️

A simple rock paper scissors game built while doing The Odin Project, to practice JS functions, conditionals and loops.

## 🔗 Live Demo

https://stoutflaco.github.io/rock-paper-scissors/

## ▶️ How to play

1. Open the live demo (or `index.html` locally)
2. Open your browser console (`Ctrl+Shift+J` / `Cmd+Option+J` on Mac)
3. Answer the prompts with `rock`, `paper`, or `scissor`
4. Best of 5 — check console for results + final score

## 🧠 What I practiced

- functions & return values
- if/else chains for win/lose/draw logic
- `Math.random()` for computer's choice
- string cleanup (`toLowerCase`, `trim`)
- loops + counters for scorekeeping

## ⚠️ Known issues

- typo = free point for computer, no retries 😭
- console-only, no on-page UI yet
